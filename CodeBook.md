# Code Book 

This code book summarizes the variables and data in the final output, `tidy_final.txt`.

## Variables

### Identifiers

* `subject` is the numerical ID of the test subject, ranging from 1 to 30.  
* `activity` is the character string representing to the type of activity performed, when the corresponding measurements were taken.  The possible values are:
    1. Walking
    2. Walking Upstairs
    3. Walking Downstairs
    4. Sitting
    5. Standing
    6. Laying

### Measurements

All variables are the mean of a measurement calculated by subject and activity. This is indicated by the initial "Mean" in the variable name. 

See <http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones> for more information.
